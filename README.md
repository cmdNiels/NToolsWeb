## Getting Started
This is an example of how you can set up your project locally.
Follow these steps to get a local copy up and running.

### Installation

1. Clone the repo
   
   ```sh
   git clone https://github.com/cmdNiels/ntools.git
   ```
2. Install NPM packages
   
   ```sh
   npm install
   ```
3. Start the project
   
   ```sh
   npm run dev
   ```
3. Build the project
   
   ```sh
   npm run build
   ```
3. Start the project
   
   ```sh
   npm run start
   ```
